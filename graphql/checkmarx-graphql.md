# Checkmarx GraphQL API

Checkmarx provides application security testing covering SAST, SCA, DAST, API security, and container security. The API covers scan management, vulnerability results, project configuration, code quality reports, and CI/CD pipeline integration.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Checkmarx REST API
**Documentation:** https://checkmarx.com/resource/documents/en/34965-128036-checkmarx-one-api.html
